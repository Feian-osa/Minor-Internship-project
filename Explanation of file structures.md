# \### **Explanation of Structure**

* *data → Stores original (`raw`) and cleaned (`processed`) datasets.*  
* *notebooks → Step-by-step Jupyter notebooks for cleaning, analysis, KPI calculation, and final dashboard.*  
* *src → Contains all reusable Python code functions for cleaning, calculations, analysis, and plotting.*  
* *reports → Stores exported figures and final report PDFs for presentations or submissions.*  
* *\*README.md → Project summary, tools used, file structure explanation for anyone visiting the repo.*
