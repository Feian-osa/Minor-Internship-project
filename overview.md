# Project Overview

The goal is to help managers quickly understand business performance and make informed decisions. The dashboard provides:

* Core KPIs (Total Sales, Total Profit, Total Orders, Average Order Value, Profit Margin)
* Sales and profit trends over time
* Category and sub-category performance
* Regional performance
* Discount vs Profit insights
* Top and bottom performing products
* 

> All analysis and visualizations are done using \*\*Python\*\* with Pandas, NumPy, Matplotlib, and Seaborn.



###  Dataset

The Superstore sales dataset contains:

* Order Date, Ship Date  
* Region / State / City  
* Category / Sub-Category  
* Product Name  
* Sales, Profit, Quantity, Discount  
* Customer Segment  



The data will be cleaned, analyzed, and visualized to provide actionable insight
